# Personal Portfolio Website

This is my personal portfolio website showcasing my UI/UX design projects, frontend development work, and software development experience.

The website is designed and developed from scratch using PHP, HTML, CSS, and JavaScript. It uses a dynamic structure to manage and display projects in a clean and reusable way.

## 🌐 Live Site

[View Portfolio](http://cleocmportfolio.infinityfreeapp.com/index.php)

## 🚀 Features

- Responsive design for desktop and mobile devices
- Dynamic project system using PHP arrays
- Reusable project detail template using GET parameters
- Embedded Figma prototypes for UI/UX projects
- Downloadable documents (technical reports, PDFs, etc.)
- Modular structure using reusable components (header/footer)
- Organized project data stored in a centralized file

## 🛠️ Built With

- HTML
- CSS
- JavaScript
- PHP

## 📁 Project Structure

/assets  
├── /css – Stylesheets  
├── /images – Project images and media  
├── /documents – PDFs (reports, resumes, etc.)

/  
├── index.php – Homepage (project list + personal info such as education & certifications)  
├── projects.php – Central project data stored in PHP arrays  
├── details.php – Dynamic project detail page using GET parameters
├── footer.php – Reusable footer component  

## 📌 Notes

This portfolio is continuously updated as I add new UI/UX and development projects. It is fully self-built without using any external portfolio templates.
